# Specification

## Summary
**Goal:** Build an AI-powered career prediction chatbot with Internet Identity authentication that provides personalized career guidance and future predictions based on user input.

**Planned changes:**
- Implement Internet Identity authentication (login/signup with passkeys or social accounts)
- Create a chat interface for authenticated users to ask career-related questions
- Develop backend logic to process career questions and generate AI-powered predictions and guidance
- Design a modern, professional frontend with welcoming aesthetics (avoiding blue/purple tones)
- Display a distinctive logo on the landing page and throughout the application
- Store chat history per authenticated user

**User-visible outcome:** Users can authenticate using Internet Identity, engage in a conversational chat interface to discuss their career ideas, and receive AI-generated future predictions and career guidance in a beautiful, professional web application.
